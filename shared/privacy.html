Política de Privacidade (Placeholder LGPD)

Este documento é um modelo inicial para revisão jurídica. O objetivo é atender os princípios da LGPD até a publicação da versão institucional final.

1. Dados Coletados

  • E-mail institucional (@tjpr.jus.br).
  • Credenciais de autenticação processadas pelo Supabase Auth.

2. Finalidade

Os dados são utilizados exclusivamente para autenticação e controle de acesso ao sistema Minutário.

3. Compartilhamento

Não há compartilhamento com terceiros além dos provedores técnicos necessários para operação da autenticação.

4. Segurança

A comunicação ocorre via HTTPS, sessões são armazenadas em mecanismos protegidos por plataforma e senhas não são persistidas em texto plano pelo aplicativo.

5. Direitos do Titular

Solicitações de acesso, correção ou exclusão de conta devem seguir o fluxo institucional do TJPR. O recurso de exclusão no app está previsto para versões futuras.
